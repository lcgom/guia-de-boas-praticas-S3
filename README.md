Este Guia de boas práticas é o resultado de um trabalho de conclusão do curso Segurança da Informação intitulado "Guia de boas práticas para o armazenamento e compartilhamento de dados no Amazon S3". No trabalho, desenvolvemos um ambiente em nuvem com integração de diversos serviços da AWS de forma a aplicar configurações seguras provindas de recomendações AWS. Por fim, a partir de análises no ambiente, elaboramos o guia.
